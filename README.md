# Interview Prep Tracker

A clean, responsive, multi-page HTML and CSS website designed for college students to track their placement preparation. This project provides a comprehensive dashboard for managing DSA progress, company preparation, mock interviews, and more.

## 🎯 Features

### Core Pages
- **Landing Page** (`index.html`) - Hero section with call-to-action buttons
- **Dashboard** (`dashboard.html`) - Main overview with progress tracking
- **DSA Tracker** (`dsa.html`) - Progress tracking for Data Structures & Algorithms
- **Company Prep** (`company.html`) - Company-wise interview preparation tracking
- **Mock Interviews** (`mocks.html`) - Mock interview performance tracking
- **HR & Tech Prep** (`prep.html`) - Preparation for technical and behavioral questions
- **Resources** (`resources.html`) - Curated learning resources and links
- **Login/Register** (`login.html`, `register.html`) - User authentication forms
- **Profile** (`profile.html`) - User profile and account management

### Key Features
- ✅ **Responsive Design** - Works perfectly on desktop, tablet, and mobile
- ✅ **Modern UI** - Clean, professional design with smooth animations
- ✅ **Progress Tracking** - Visual progress bars and completion indicators
- ✅ **Company Focus** - Track preparation for specific companies
- ✅ **Mock Interview Management** - Rate and review mock interviews
- ✅ **Resource Library** - Curated links to learning resources
- ✅ **No JavaScript Required** - Pure HTML and CSS implementation

## 🎨 Customization

### Colors
Modify the CSS variables in `css/style.css`:
```css
:root {
    --primary-color: #3A86FF;
    --accent-color: #FFBE0B;
    --background-color: #F9F9F9;
    /* ... other variables */
}
```

### Fonts
Change the Google Fonts import in `css/style.css`:
```css
@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@300;400;500;600;700&family=Inter:wght@300;400;500;600;700&display=swap');
```
